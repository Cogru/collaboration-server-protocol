---
title: room::delete_file
---

Request to delete a file.

```json
{
    "method": "room::delete_file",
    "file": <path>,
}
```

- `file` is the path to the file you want to delete.

<details>
<summary>Example response</summary>

```json
{
    "method": "room::delete_file",
    "file": <path>,
    "status": "success",
}
```

</details>
